Project Overview

Transform your Ikea RÖDALM picture frame into a smart eInk display with this custom-designed 3D-printable insert.

This product provides precisely fitted mounting parts for integrating a 7.5" Waveshare e-Paper display into the popular Ikea RÖDALM 5"x7" (13x18cm) picture frame. The insert holds all electronic components securely in place while maintaining the elegant look of the frame.

With the provided STL files, you can easily create your own frame insert on your 3D printer and build a battery-powered eInk display for your home.

Product Information

This product provides 3D-printable mounting and assembly parts for building an eInk picture frame using the Ikea RÖDALM frame.

How to Use

Step-by-Step Guide

  1. Gather Your Materials
    • Purchase all components from the "Bill of Materials" section above
    • All Amazon links are pre-configured with affiliate tags to support Nerdiy.de development
    • For STL files, purchase through Nerdiy.de Shop to support independent makers
  2. Download 3D Files
    • Download from Nerdiy.de Shop (recommended - supports independent makers)
    • Alternative: Download from Printables
  3. ESPHome Firmware Configuration
    • The required ESPHome firmware configuration is included in the esphome_config/ folder
    • Flash the ESP32 with the provided configuration before assembly
    • Customize settings as needed for your specific setup
  4. Prepare for 3D Printing
    • Recommended print settings:
    • Material: PLA, PETG, or similar
    • Layer height: 0.2mm
    • Infill: 20-30%
    • Supports: As needed
  5. Assembly
    • Follow the detailed assembly instructions included with the STL files
    • Install thread inserts for secure mounting
    • Connect all electronic components
    • Test the display before final assembly
  6. Final Installation
    • Insert the completed assembly into your Ikea RÖDALM frame
    • Secure all components
    • Set up your eInk display software
