# 6x Safety Switch desk mounting board for XIAO ESP32-S3 by Nerdiy.de

---

## 🎯 Project Overview

A professional desk mounting solution for controlling up to six safety switches using a XIAO ESP32-S3 microcontroller.

This 3D-printable mounting board provides a clean and organized way to integrate multiple safety switches into your workspace. Perfect for controlling power tools, 3D printers, or other equipment that requires physical safety cutoffs. The integrated XIAO ESP32-S3 enables smart features like remote monitoring, automation, and integration with home automation systems through ESPHome.

With the provided STL files, you can easily create your own professional-grade safety switch control board.

## 🎯 How to Use

### Step-by-Step Guide

1. **Gather Your Materials**
   - Review the complete bill of materials on the Printables product page
   - Acquire all necessary safety switches and electronic components
   - For STL files, [purchase through Nerdiy.de Shop](https://www.nerdiy.de/) to support independent makers

2. **Download 3D Files**
   - [🛍️ Download from Nerdiy.de Shop](https://www.nerdiy.de/) (recommended - supports independent makers)
   - Alternative: [Download from Printables](https://www.printables.com/model/1567337-6x-safety-switch-desk-mounting-board-for-xiao-esp3)
   - Alternative: [Download from Cults3D](https://cults3d.com/de/modell-3d/gadget/6x-safety-switch-desk-mounting-board-for-xiao-esp32-s3-by-nerdiy-de)

3. **ESPHome Firmware Configuration**
   - The required ESPHome firmware configuration is included in the `esphome_config/` folder
   - Flash the XIAO ESP32-S3 with the provided configuration before assembly
   - Customize switch assignments and automation rules as needed for your specific setup

4. **Prepare for 3D Printing**
   - Recommended print settings:
     - Material: PLA, PETG, or ABS (for higher temperature resistance)
     - Layer height: 0.2mm
     - Infill: 30-50% (higher for mechanical stability)
     - Supports: As needed

5. **Assembly**
   - Follow the detailed assembly instructions included with the STL files or on the Printables page
   - Install all safety switches securely
   - Wire the switches to the XIAO ESP32-S3 according to the wiring diagram
   - Test each switch individually before final installation

6. **Final Installation**
   - Mount the completed board to your desk or workbench
   - Connect to your equipment following proper electrical safety procedures
   - Configure your home automation system if desired
   - **Always test the emergency stop functionality before connecting critical equipment**
